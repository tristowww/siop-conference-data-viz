# 2026 Raw Public Agenda Extracts

These files are public SIOP 2026 conference-program extracts from the Whova embedded agenda. They
do not use authenticated attendee access, private messages, comments, or attendee profile data.

- `whova_public_embed_sessions.csv`: compact public embedded-agenda scrape.
- `whova_public_agenda_api_sessions.csv`: richer public agenda API extract used for the final
  2026 session file.

The analysis-ready version lives in `data/processed/sessions_2026_enriched.csv`.
